# AR访问windows共享文件夹

## 问题描述

- 客户需要在AR的环境中，定时把生产数据文件发送到Windows服务器的共享文件夹中。
- 即AR环境需要访问Windows的共享目录。

## 解决方法

Windows 配置的共享文件夹可以通过 CIFS 的方式进行，搜索 AS4.5.2.10 Help ID 14.4.3.2.11.30
