## 2.1 官方例程

- 在AS软件中，在Logical View中右键 Add Object，可找到Samples，其中有大量的通信案例Demo程序以供参考使用。
- <span style="background:#A0CCF6">1___</span>右键Logical View，勾选Samples，双击选择 Library Samples。

- <span style="background:#A0CCF6">2___</span>可见大量例程库资源可供选择
