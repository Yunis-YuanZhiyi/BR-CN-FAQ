# X20BC0063 Profibus DP总线控制器后最多可以连接多少X20模块
需要使用官方网站上下载的计算工具excel表格配置相应IO模块才能明确是否可以在X20BC0063后组态。

使用方法：在Used中填入该模块欲使用的数量后，可以在available中检查剩余可配置数量，在Status中可检查Profibus DP数据帧是否能容纳这么多模块进行通信。
