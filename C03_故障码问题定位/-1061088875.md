# 报错内容

- -1061088875: Connection to configured mapp server failed. Url: "a" Status: "b"
- AS Help中报错信息
- 描述
    - 尝试连接mapp服务器失败。
- 原因和解决方案
    - 检查服务器是否正在运行/检查服务器是否配置正确/查看状态

# 遇到常见场景

- 使用mappAlramX结合mappView，功能异常。
- AlarmList 部件不显示警报
- Logger中记录 -1061088875

# 解决方式

- 手动添加 MpServer 库
