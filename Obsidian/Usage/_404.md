# 网络问题 → 刷新一下

- 较新的内容，可能需要刷新一下才能看到。
- 在一些特殊网络状态下可能访问不了，例如PulseSecure链接的内网，可尝试切换网络。

# 链接错误 → 检查复制链接

- 链接错误，输入漏了相关信息
- 原链接的文件名被修改，对应链接被改了
- 可根据链接中包含的索引信息，例如B08 067，在 www.brhelp.cn 中直接目录查找
