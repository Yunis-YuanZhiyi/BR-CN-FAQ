# 059AS中插入X20IF1043-1模块后编译报错
# 现象
- AS中插入X20IF1043-1模块，编译即报错Could not export configuration data of interface 'IF1' on device 'X20IF1043_1'.
# 原因
- 1.6版本DTM中X20IF1043-1模块描述信息文件中包含德文字符

# 解决方式 
- 从B&R官网下载安装1.5版本DTM并替换安装
