# 006打开Automation Studio项目时，出现中文注释乱码的情况怎么办？
Tags：注释、乱码
A：
出现这种情况的原因可能是因为原项目是在Windows 7的环境下进行编写的。当操作系统升级至Windows 10时，打开项目时就会出现中文注释乱码。
