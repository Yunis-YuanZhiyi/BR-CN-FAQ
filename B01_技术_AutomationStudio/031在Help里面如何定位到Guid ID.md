# 在Help里面如何定位到Guid ID?
在页面任意出右键，选择属性，就可以看到Guid ID

![Img](./FILES/026在Help里面如何定位到Guid%20ID.md/img-20220615135324.png)

在Help 的主菜单中点击View，然后在下拉菜单中选中Goto Page。会有一个弹出框，在弹出框中输入Guid ID即可。
